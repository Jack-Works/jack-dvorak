# jack-dvorak

A programmer dvorak variant I use.

This is a mixed layout of [Programmer Dvorak] and [macOS's Dvorak - QWERTY âŒ˜](https://en.wikipedia.org/wiki/Dvorak_keyboard_layout#Mac_OS:~:text=Apple%20also%20includes%20a%20Dvorak%20variant%20they%20call%20%22Dvorak%C2%A0%E2%80%93%20Qwerty%20%E2%8C%98%22.%20With%20this%20layout%2C%20the%20keyboard%20temporarily%20becomes%20QWERTY%20when%20the%20Command%20(%E2%8C%98/Apple)%20key%20is%20held%20down.) layout.

I also swapped some symbols to fit TypeScript (`;:`, `-_`, `/?`, `\|`, <code>`#</code>).
